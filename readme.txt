WooCommerce integration for MiFinity payments gateway. Accept credit cards directly on WordPress e-commerce websites.

== Description ==

[MiFinity Payments Gateway](https://www.mifinity.com/business/mifinity-payments-gateway/) allows you to accept credit cards from your customers from all over the world on your websites and settles funds automatically to your merchant account.

[WooCommerce](https://woocommerce.com/) is one of the popular and most powerful e-commerce solutions for WordPress. This platform is widely supported within the WordPress community which makes for simple plug-and-play installation.

* Active MiFinity merchant account.
* [**WooCommerce**](https://woocommerce.com/)  version 3.3.0 or later.

== Installation ==

1. Activate the plugin (WordPress -> Plugins).
2. Go to the WooCommerce settings page (WordPress -> WooCommerce -> Settings) and select the Payments tab.
3. Under the Payments tab, you will find all the available payment methods. Find the 'MiFinity Payment Gateway' link in the list and click it.
4. On this page you will find all of the configuration options for this payments gateway and you need to enable the method by using the checkbox.
5. To obtain your MiFinity account details (API Key, account number) please contact sales@mifinity.com and MiFinity will help you setup your account and assist you to start processing transactions. 

#### Technical Requirements
The plugin requires the following mandatory fields to be present on the WooCommerce billing details form:
* First name
* Last name
* Country
* Address
* Postcode
* City
* Phone number
* Email address

You are now connected to WooCommerce and ready to accept credit cards via the MiFinity Payments Gateway.
